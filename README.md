## Привет!
Основные инструменты: `Java, Lua`

Дополнительный опыт (сортировка по опыту): `Unity, C#, JavaScript, HTML/CSS, C++`

---

**Опыт и направления:**
*   **Java:** Сетевые, графические и аудио движки, моддинг старых версий Minecraft, менеджер паролей, ассемблер и так далее. Использую язык как универсальный инструмент для реализации сложных проектов.

    Примеры:
    
    - Менеджер паролей [Passer](https://github.com/Onran0/Passer)
    
    - Библиотека [JSW](https://github.com/Onran0/JSW) для удобной работы с форматом .structure
    
    - Ассемблер [medpow8](https://github.com/Onran0/medpow8-assembler) для собственной архитектуры ЦП
    
    - [spookylauncher](https://github.com/spookylauncher/spookylauncher) для удобной игры в мистические версии Minecraft

*   **Lua:** скриптинг и логика.

    Примеры:
    - Окружение [VoxelCore](https://github.com/MihailRis/voxelcore) для контент-паков и мои проекты для него:
        - [kcp2vc](https://github.com/Onran0/kcp2vc) для работы с протоколом **kcp**
        - [VCcURL](https://github.com/Onran0/VCcURL) для работы с **cURL**
        - [VCLibPNG](https://github.com/Onran0/VCLibPNG) для **I/O** с **PNG**

---

Предпочитаю писать код своими руками. В разработке использую ИИ только для быстрой справки, чтобы задавать качество и архитектуру своих проектов.
